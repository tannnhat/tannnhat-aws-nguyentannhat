## Chi tiết Lab đã thực hành

---

### Lab 42 – AWS CodeCommit

#### 1. Quản lý Source Code

- Tạo CodeCommit Repository.
- Clone Repository về máy.
- Commit và Push mã nguồn.
- Quản lý các phiên bản mã nguồn.

**Hiểu:**

- Quy trình quản lý Source Code trên AWS.
- Làm việc với Git và CodeCommit.

---

### Lab 39 – AWS CodePipeline & CodeBuild

#### 1. Tạo CI/CD Pipeline

- Tạo CodeBuild Project.
- Cấu hình Build Specification.
- Tạo AWS CodePipeline.
- Kiểm tra quá trình Build và Deploy.

**Hiểu:**

- Quy trình Continuous Integration.
- Tự động Build và triển khai ứng dụng.

---

### Lab 45 – AWS CodeDeploy

#### 1. Tự động triển khai ứng dụng

- Tạo Deployment Application.
- Tạo Deployment Group.
- Cấu hình Deployment.
- Kiểm tra trạng thái triển khai.

**Hiểu:**

- Continuous Deployment trên AWS.
- Tự động cập nhật ứng dụng.

---

### Module thực hành – Monitoring Pipeline

- Theo dõi Pipeline Execution.
- Kiểm tra Build Logs.
- Xử lý lỗi khi Build hoặc Deploy thất bại.

**Hiểu:**

- Quy trình giám sát CI/CD.
- Phân tích và khắc phục lỗi Pipeline.

## Kết quả đạt được:

- Hiểu quy trình DevOps và CI/CD trên nền tảng AWS.
- Thành thạo quản lý mã nguồn bằng AWS CodeCommit.
- Xây dựng thành công quy trình Build tự động với AWS CodeBuild.
- Tạo và vận hành CI/CD Pipeline bằng AWS CodePipeline.
- Triển khai ứng dụng tự động bằng AWS CodeDeploy.
- Biết cách theo dõi Build Logs và Pipeline Execution để xử lý sự cố.
- Nắm được quy trình tự động hóa triển khai phần mềm từ Source Code đến Production.
- Hoàn thành các bài thực hành về AWS DevOps Services, tạo nền tảng cho việc xây dựng hệ thống triển khai tự động trong các dự án thực tế.
